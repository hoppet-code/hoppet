HOPPET documentation
=====================

- manual/: The HOPPET manual in LaTeX format, including the v2 update note. Run
  `latexmk -pdf HOPPET-doc.tex` in that directory to generate the PDF 
  from the source.

- python-auto/: skeleton for automatic generation of documentation for
  the Python interface. Run `make html` in that directory to generate the
  HTML documentation from the source.
